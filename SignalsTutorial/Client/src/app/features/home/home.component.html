I.T. Associates Assignment

Using the attached client JSON data, create a simple angular application based on the instructions below.

  1. Create a client module.
  2. Create a list component to display the client data. This component should be created as a reusable component.
  3. Display the list of clients.
  4. Highlight the rows where the client type is media.
  5. Create a simple edit form component to allow users to create or update client data.
  6. The client type control should be a dropdown with the following options "Media", "Finance" and "Catering".
  7. Create an auto-complete input component where it could be able to reuse across the application to select the line of business.
  8. The “client name” textbox should be a mandatory field.
  9. When the user saves the client, the application should be redirected back to the list page and highlight the edited item using a state management option (Angular services, RxJS or NgRx).
